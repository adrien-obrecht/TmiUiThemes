# TMI Ui Themes

TMI angelscript plugin providing a bank of 40+ ImGui themes, taken from the [ImThemes](https://github.com/Patitotective/ImThemes) repository.

![](https://github.com/adrien-obrecht/TmiUiThemes/blob/main/demo.gif)

# Download

Simply download the zip file [here]([https://github.com/adrien-obrecht/TmiUiThemes/releases/tag/Release](https://github.com/adrien-obrecht/TmiUiThemes/releases/download/V1.1.0/UiThemes.zip)) and dezip it inside your `TMInterface/Plugins` folder.

# Adding a theme / contributing

All of the themes data originate from the `themes.toml` file. To add / modify a theme, simply modify this file, then generate the angelscript plugin using the `parse_themes.py` python script. Feel free to submit a PR !
